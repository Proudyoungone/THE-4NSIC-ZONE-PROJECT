DFIR Command Playbook
System Identification & Time
hostname whoami ver systeminfo time /t date /t tzutil /g
Users & Logons
net user net user username net localgroup administrators quser qwinsta
Processes & Memory
tasklist tasklist /v tasklist /svc wmic process list full wmic process get caption,commandline,processid
Services & Persistence
sc query sc query type= service sc queryex wmic service list full
Startup & Autoruns
reg query HKLM\Software\Microsoft\Windows\CurrentVersion\Run reg query HKCU\Software\Microsoft\Windows\CurrentVersion\Run dir "C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup" dir "%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup"
Network Forensics
ipconfig /all arp -a route print netstat -ano netstat -abno nbtstat -n nbtstat -a IP
Firewall, Shares & Lateral Movement
netsh advfirewall show allprofiles netsh advfirewall firewall show rule name=all net use net share
File System Forensics
dir /a /s C:\ > files.txt tree C:\ /f /a attrib fsutil fsinfo drives fsutil dirty query C:
Event Log Triage
doskey /history wevtutil qe Security /f:text /c:50 wevtutil qe System /f:text /c:50
USB & External Devices
reg query HKLM\SYSTEM\CurrentControlSet\Enum\USBSTOR reg query HKLM\SYSTEM\MountedDevices
Malware Triage & Scheduled Tasks
assoc ftype schtasks schtasks /query /fo LIST /v
Prefetch (Execution Evidence)
dir C:\Windows\Prefetch
Shadow Copies & Backups
vssadmin list shadows vssadmin list volumes
Disk & Partitions
diskpart list disk list volume detail volume
File Hashing
certutil -hashfile evidence.img MD5 certutil -hashfile evidence.img SHA256
Base64 Encoding & Staging Abuse
certutil -decode input.txt output.exe certutil -encode file.exe encoded.txt
Deep Registry Hunting
reg query HKLM /s reg query HKCU /s reg query HKLM\Software\Microsoft\Windows\CurrentVersion
Script & Ransomware Indicators
assoc | find ".js" assoc | find ".vbs" assoc | find ".ps1"
Time Synchronization
w32tm /query /status
EVTX Log Export
wevtutil epl Security security.evtx wevtutil epl System system.evtx
Drivers & Rootkits
driverquery driverquery /v driverquery /si
Environment Variables & Staging
set echo %TEMP% echo %APPDATA%
IOC & Keyword Searching
where file.exe findstr /i "password" *.txt findstr /s /i "token" C:\*
Live Containment
taskkill /PID 1234 /F taskkill /IM malware.exe /F
System Integrity
sfc /scannow chkdsk C: /f
